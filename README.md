## 📌 Descripción

Este proyecto forma parte de la estrategia **Fábrica Escuela**, en la cual los estudiantes asumen roles dentro de un equipo de desarrollo de software utilizando la metodología **Scrum**.

El sistema consiste en una plataforma web orientada a la **gestión de finanzas personales**, permitiendo a los usuarios registrar, organizar y analizar su información financiera para tomar decisiones más informadas.

---

## 🎯 Contexto del problema

Muchas personas presentan dificultades al momento de comprender sus hábitos financieros, lo que impacta directamente en su capacidad de ahorro y planificación económica.

Actualmente, existe la necesidad de herramientas que permitan:
- Visualizar ingresos y gastos de forma clara  
- Entender patrones de consumo  
- Tomar decisiones financieras más conscientes  

Este proyecto busca dar solución a esta problemática mediante una plataforma accesible y estructurada.

---

## 🧩 Funcionalidades principales

El sistema permitirá a los usuarios:

- 💰 Registrar ingresos y gastos personales  
- 🏷️ Clasificar transacciones por categorías  
- 📊 Gestionar presupuestos mensuales  
- ⚖️ Calcular balances financieros  
- 📈 Generar reportes sobre hábitos de consumo  
- 💡 Recibir recomendaciones básicas para mejorar su salud financiera
